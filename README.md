## 🚀 Возможности

* **Управление данными:** Добавление, редактирование и удаление транзакций.
* **Визуализация:** Построение круговых и линейных диаграмм по категориям расходов.
* **Экспорт данных:** Сохранение отчетов в формате PDF.
* **Настройки:** Гибкая настройка валют и категорий.
